什么是Appecker？
------------------------------------------

Appecker是一款专门为 i0S系统系统定制的的自动化测试框架
v1.0.0版本仅对i0S6.0 , 7.0 32位提供支持。


如何使用？
------------------------------------------
可以从代码目录的DemoProject开始学习该组件

相关文档保存在文件夹doc中

可以参考文档《Appecker 集成说明.doc》 
如果需要使用更多SKD，请参考《Appecker SDK说明.doc》
